**End Palette**

Stop guessing color scales. End Palette generates production-ready color scales from a single hex code — built on the systems you already use.

**Features**

* **Generate from any color** — Enter a hex code, get a full scale instantly.
* **Industry standards built-in** — Tailwind, Material, Ant Design, Radix, Carbon, Bootstrap.
* **Official palettes** — Browse and insert palettes from GitHub, Shopify, Atlassian, and more.
* **Personal library** — Save palettes across projects.
* **One-click insert** — Drop a formatted swatch frame onto your canvas, ready for variables.

**How to use**

1. Enter a base hex color
2. Pick a system (Tailwind, Material, etc.)
3. Hit Generate
4. Insert to canvas or save to library
